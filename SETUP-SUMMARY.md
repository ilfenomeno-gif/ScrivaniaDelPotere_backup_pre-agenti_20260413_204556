### Passo 1: Leggi la Documentazione (5 min)

2. Apri **.claude/README.md** - Overview del sistema

### Passo 2: Prepara il Tuo Brand (10-30 min)
Crea file nella cartella **_context/**:
- `brand-guidelines.md` - Colori, font, tone of voice
- `visual-identity.md` - Logo, palette, stile visuale

### Passo 3: Attiva il Sistema (1 min)
Scegli uno scenario:

**Opzione A: Campagna Completa (Consigliato)**
```
@orchestrator → Gestisci la campagna [Nome]:
- Obiettivi: [Goals]
- Budget: [Budget]
- Timeline: [Duration]
- Target: [Audience]
- Brand context: /_context/brand-guidelines.md
```

**Opzione B: Step-by-Step**
```
1. @market-researcher → Analizza mercato per [segmento]
2. @campaign-strategist → Crea strategia basata su market intel
3. @content-creator → Genera copy usando content brief
4. @creative-designer → Progetta visual usando creative brief
5. @data-analyst → Monitora performance post-launch
```

### Passo 4: Monitora Progress
- Orchestrator fornisce daily status reports
- Data Analyst fornisce performance dashboard
- Task board Notion traccia lo stato

## 🎓 Prompts di Base (Copia-Incolla)

### Prompt 1: Campagna Completa
```
@orchestrator → Gestisci la campagna [Campaign Name]:
- Obiettivi: [Goals]
- Budget: [Total Budget]
- Timeline: [Duration in weeks]
- Target: [Audience Description]
- Brand context: /_context/brand-guidelines.md
```

### Prompt 2: Market Research Only
```
@market-researcher → Analizza il mercato per [Segment]:
- Mercato: [Geographic/Segment]
- Focus: Competitor analysis, trend, audience
- Output: Report + personas + positioning
```

### Prompt 3: Strategia Only
```
@campaign-strategist → Crea strategia per [Campaign]:
- Obiettivi: [Goals]
- Budget: [Budget]
- Timeline: [Duration]
- Output: Strategy doc + content brief + creative brief
```

### Prompt 4: Content Only
```
@content-creator → Genera contenuti per [Campaign]:
- Channel: [social/email/landing/ads]
- Topic: [Topic]
- Tone: [Tone description]
- Include A/B variations: Yes
```

### Prompt 5: Design Only
```
@creative-designer → Crea design per [Campaign]:
- Channel: [social/email/landing/ads]
- Topic: [Topic]
- Style: [Style description]
- Include mockup: Yes
```

### Prompt 6: Data Analysis Only
```
@data-analyst → Analizza performance di [Campaign]:
- KPI: [KPI list]
- Period: [Date range]
- Include: A/B analysis, forecast, recommendations
```
