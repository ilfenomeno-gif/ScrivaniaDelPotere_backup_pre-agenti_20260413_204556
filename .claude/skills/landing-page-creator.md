# Skill: landing-page-creator

## Descrizione
Genera landing page marketing personalizzate a partire da brief, obiettivi e contesto brand.

## Flusso di lavoro
1. Ricevi brief, obiettivi e parametri (target, offerta, canale, ecc.).
2. Recupera linee guida brand da _context/ e template da _templates/.
3. Genera struttura, copy e visual coerenti con il brand.
4. Restituisce file HTML/CSS e, se richiesto, specifiche di design.

## Brand-agnostic
La skill non contiene riferimenti a brand specifici. Recupera il contesto brand a runtime.

## Output
- File HTML/CSS della landing page
- Specifiche di design (opzionale)

---

### Esempio di prompt per l'utente
"Crea una landing page per la campagna Q3 lancio estivo, target 25-40 anni, offerta HIIT Outdoor."
