# Skill: deck-template-patcher

## Descrizione
Applica patch e correzioni a template di presentazione esistenti, aggiornando colori, font, layout o elementi visivi secondo nuove specifiche.

## Flusso di lavoro
1. Ricevi template pptx e specifiche di patch (colori, font, layout, elementi visivi).
2. Applica le modifiche richieste mantenendo la coerenza con il brand.
3. Restituisce il template aggiornato nella cartella di output.

## Brand-agnostic
La skill non contiene riferimenti a brand specifici. Recupera il contesto brand a runtime.

## Output
- Template pptx aggiornato

---

### Esempio di prompt per l'utente
"Applica una patch al template /_templates/deck_template.pptx aggiornando i colori secondo le nuove linee guida."
