# Skill: deck-template-exporter

## Descrizione
Esporta template di presentazione e deck in diversi formati (pptx, pdf, immagini) per la distribuzione e l'utilizzo cross-platform.

## Flusso di lavoro
1. Ricevi file pptx o deck da esportare e parametri di output (formato, qualità, pagine, ecc.).
2. Esporta nei formati richiesti (pptx, pdf, png, jpg, ecc.).
3. Restituisce i file esportati nella cartella di output.

## Brand-agnostic
La skill non contiene riferimenti a brand specifici. Recupera il contesto brand a runtime.

## Output
- File esportati (pptx, pdf, png, jpg, ecc.)

---

### Esempio di prompt per l'utente
"Esporta il deck /output/strategia_Q3.pptx in formato PDF e PNG."
