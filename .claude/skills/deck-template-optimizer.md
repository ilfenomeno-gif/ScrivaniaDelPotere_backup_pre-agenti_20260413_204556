# Skill: deck-template-optimizer

## Descrizione
Ottimizza template di presentazione e deck per performance, accessibilità e coerenza visiva.

## Flusso di lavoro
1. Ricevi file pptx/deck e parametri di ottimizzazione (performance, accessibilità, coerenza).
2. Applica ottimizzazioni su colori, font, layout, elementi visivi e struttura file.
3. Restituisce il file ottimizzato e un report delle modifiche.

## Brand-agnostic
La skill non contiene riferimenti a brand specifici. Recupera il contesto brand a runtime.

## Output
- File ottimizzato (pptx)
- Report ottimizzazione (md, txt)

---

### Esempio di prompt per l'utente
"Ottimizza il deck /output/strategia_Q3.pptx per accessibilità e performance."
