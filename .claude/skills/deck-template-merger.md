# Skill: deck-template-merger

## Descrizione
Unisce più template di presentazione o deck in un unico file, mantenendo coerenza di colori, font, layout e stile.

## Flusso di lavoro
1. Ricevi i file pptx/deck da unire e parametri di output.
2. Analizza e armonizza colori, font, layout e stile.
3. Genera il file unificato e lo salva nella cartella di output.

## Brand-agnostic
La skill non contiene riferimenti a brand specifici. Recupera il contesto brand a runtime.

## Output
- File deck unificato (pptx)

---

### Esempio di prompt per l'utente
"Unisci i deck /output/strategia_Q3.pptx e /output/analisi_audience.pptx in un unico file."
