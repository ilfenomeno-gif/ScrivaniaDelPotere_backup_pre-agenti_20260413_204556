# Skill: deck-template-localizer

## Descrizione
Localizza template di presentazione e deck in diverse lingue e mercati, adattando testi, formati e riferimenti culturali.

## Flusso di lavoro
1. Ricevi file pptx/deck e lingua/mercato di destinazione.
2. Traduce testi, adatta formati e riferimenti culturali secondo le linee guida brand.
3. Restituisce il file localizzato nella cartella di output.

## Brand-agnostic
La skill non contiene riferimenti a brand specifici. Recupera il contesto brand a runtime.

## Output
- File localizzato (pptx)

---

### Esempio di prompt per l'utente
"Localizza il deck /output/strategia_Q3.pptx in inglese per il mercato UK."
