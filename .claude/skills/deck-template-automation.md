# Skill: deck-template-automation

## Descrizione
Automatizza la generazione, validazione, esportazione e archiviazione di template e deck secondo flussi predefiniti.

## Flusso di lavoro
1. Ricevi richiesta di automazione (es. genera, valida, esporta, archivia deck).
2. Esegue le operazioni richieste in sequenza, utilizzando le skill appropriate.
3. Restituisce log delle operazioni e output prodotti.

## Brand-agnostic
La skill non contiene riferimenti a brand specifici. Recupera il contesto brand a runtime.

## Output
- Log automazione (md, txt)
- Output prodotti (pptx, pdf, ecc.)

---

### Esempio di prompt per l'utente
"Automatizza la generazione, validazione ed esportazione del deck strategia_Q3.pptx."
