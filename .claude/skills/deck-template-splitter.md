# Skill: deck-template-splitter

## Descrizione
Divide un template di presentazione o un deck in più file, secondo criteri di layout, sezioni o contenuti.

## Flusso di lavoro
1. Ricevi file pptx/deck e criteri di suddivisione (layout, sezioni, slide, ecc.).
2. Divide il file secondo i criteri richiesti.
3. Restituisce i file suddivisi nella cartella di output.

## Brand-agnostic
La skill non contiene riferimenti a brand specifici. Recupera il contesto brand a runtime.

## Output
- File deck suddivisi (pptx)

---

### Esempio di prompt per l'utente
"Dividi il deck /output/strategia_Q3.pptx in file separati per ogni sezione."
