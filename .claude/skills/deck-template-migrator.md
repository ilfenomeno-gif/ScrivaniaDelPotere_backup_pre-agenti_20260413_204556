# Skill: deck-template-migrator

## Descrizione
Migra template di presentazione tra diversi sistemi o versioni, adattando colori, font, layout e elementi visivi secondo le nuove specifiche richieste.

## Flusso di lavoro
1. Ricevi template pptx di origine e specifiche di destinazione.
2. Adatta colori, font, layout e elementi visivi secondo le nuove specifiche.
3. Restituisce il template migrato nella cartella di output.

## Brand-agnostic
La skill non contiene riferimenti a brand specifici. Recupera il contesto brand a runtime.

## Output
- Template migrato (pptx)

---

### Esempio di prompt per l'utente
"Migra il template /_templates/deck_template_v1.pptx alle nuove specifiche in /_context/brand-guidelines-v2.md."
