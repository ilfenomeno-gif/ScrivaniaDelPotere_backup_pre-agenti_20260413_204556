# Skill: deck-template-creative-optimizer

## Descrizione
Ottimizza template e deck dal punto di vista creativo, migliorando impatto visivo, originalità e coerenza con il brief.

## Flusso di lavoro
1. Ricevi file pptx/deck e brief creativo.
2. Analizza punti di forza e debolezza creativa.
3. Applica ottimizzazioni su visual, layout, elementi grafici e copy.
4. Restituisce il file ottimizzato e un report delle modifiche creative.

## Brand-agnostic
La skill non contiene riferimenti a brand specifici. Recupera il contesto brand a runtime.

## Output
- File ottimizzato (pptx)
- Report ottimizzazione creativa (md, txt)

---

### Esempio di prompt per l'utente
"Ottimizza creativamente il deck strategia_Q3.pptx per maggiore impatto e originalità."
