# Skill: creative-designer

## Descrizione
Progetta visual, layout e asset grafici per campagne e contenuti marketing.

## Flusso di lavoro
1. Ricevi brief, asset e linee guida brand a runtime.
2. Progetta visual coerenti con il brand.
3. Restituisce file grafici (PNG, SVG, PSD, ecc.).

## Brand-agnostic
La skill non contiene riferimenti a brand specifici. Recupera il contesto brand a runtime.

## Output
- File grafici (PNG, SVG, PSD, ecc.)

---

### Esempio di prompt per l'utente
"Progetta una serie di visual per la campagna Q3 lancio estivo, stile energico e moderno."
